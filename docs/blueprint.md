# **App Name**: ScrollMotion

## Core Features:

- Media Expansion on Scroll: Expands a video or image as the user scrolls, creating an immersive experience.
- Dynamic Text Animation: Animates text elements to move in opposite directions based on scroll progress, enhancing visual appeal.
- Mobile-Responsive Design: Adjusts the expansion and animation parameters based on screen size to ensure optimal viewing on mobile devices.
- YouTube Video Support: Plays YouTube videos seamlessly, with autoplay, looping, and muted settings, and applies a subtle darkening overlay.
- Image and Video Flexibility: Supports both image and video content, adapting the display and effects accordingly.

## Style Guidelines:

- Primary color: Soft, muted blue (#A0D2EB) to evoke a sense of calmness and immersion.
- Background color: Very light, desaturated blue (#F0F8FF), almost white, for a clean and unobtrusive backdrop.
- Accent color: Pale yellow (#FAEBD7) for highlighting key text elements and providing subtle contrast.
- Headline font: 'Belleza' sans-serif, with a personality aligned to fashion, art, and design. Body font: 'Alegreya' serif. Note: currently only Google Fonts are supported.
- Full-screen sections with centered content to maximize the immersive scrolling experience.
- Smooth, subtle transitions and animations triggered by scroll events to enhance the interactive feel.
- Minimalist icons to complement the clean and modern design.
